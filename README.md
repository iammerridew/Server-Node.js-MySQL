# Server with Node.js and MySQL

REST API built with Node.js (express, morgan, cors) and MySQL. 

## Installation

Import and configure the database in /src/database.ts.
Install the dependencies necessary for its operation with:

```bash
npm i
```

Build the app with: 

```bash
npm run build
```

Run the server with:

```bash
node dist/index.js
```
